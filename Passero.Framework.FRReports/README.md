# Passero.Framework.FRReports

Libreria per la gestione di report FastReport nel Passero Framework.

## Descrizione

Passero.Framework.FRReports fornisce un'integrazione completa con FastReport OpenSource per la creazione, gestione ed esportazione di report all'interno del Passero Framework, con supporto multi-target per .NET Framework e .NET moderno.

## Target Framework

- .NET Framework 4.8
- .NET 8.0

## Dipendenze Principali

- **FastReport.OpenSource** - Motore di reporting open source
- **FastReport.Data.MsSql** - Provider dati per SQL Server
- **FastReport.OpenSource.Export.PdfSimple** - Esportazione report in formato PDF
- **Wisej-3** - Framework per applicazioni web progressive
- **Dapper** - Micro ORM per accesso ai dati

## Installazione

```bash
dotnet add package Passero.Framework.FRReports
```

Oppure tramite NuGet Package Manager:

```powershell
Install-Package Passero.Framework.FRReports
```
