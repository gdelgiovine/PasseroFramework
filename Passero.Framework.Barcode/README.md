# Passero.Framework.Barcode

Libreria di supporto per barcode per Passero Framework.

## Descrizione

Passero.Framework.Barcode fornisce funzionalità complete per la generazione e la lettura di barcode all'interno del Passero Framework, con supporto multi-target per .NET Framework e .NET moderno.

## Target Framework

- .NET Framework 4.7
- .NET Framework 4.8
- .NET 8.0

## Dipendenze

- **ZXing.Net** - Libreria per generazione e lettura di barcode
- **QRCoder** - Generazione di codici QR
- **FreeSpire.Barcode** - Supporto aggiuntivo per barcode
- **Microsoft.VisualBasic** - Supporto runtime

## Installazione

```bash
dotnet add package Passero.Framework.Barcode
```

Oppure tramite NuGet Package Manager:

```powershell
Install-Package Passero.Framework.Barcode
```
