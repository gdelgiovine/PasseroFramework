# Passero.Framework.SSRSReports

Libreria per la gestione di report Microsoft SSRS nel Passero Framework.

## Descrizione

Passero.Framework.SSRSReports fornisce un'integrazione completa con Microsoft SQL Server Reporting Services (SSRS) per la creazione, gestione e visualizzazione di report all'interno del Passero Framework, con supporto multi-target per .NET Framework e .NET moderno.

## Target Framework

- .NET Framework 4.8
- .NET 8.0

## Dipendenze Principali

- **ReportViewerCore.NETCore** (per .NET 8.0) - Visualizzatore report per .NET Core
- **Microsoft.ReportingServices.ReportViewerControl.WebForms** (per .NET Framework 4.8) - Controllo ReportViewer per WebForms
- **Microsoft.SqlServer.Types** - Tipi spaziali SQL Server
- **Wisej-3** - Framework per applicazioni web progressive
- **Dapper** - Micro ORM per accesso ai dati
- **MiniExcel** - Gestione file Excel

## Installazione

```bash
dotnet add package Passero.Framework.SSRSReports
```

Oppure tramite NuGet Package Manager:

```powershell
Install-Package Passero.Framework.SSRSReports
```
