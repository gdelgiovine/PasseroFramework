# Passero.Framework.Controls

Libreria di controlli UI per Passero Framework.

## Descrizione

Passero.Framework.Controls fornisce una collezione completa di controlli UI personalizzati per applicazioni basate su Wisej.NET all'interno del Passero Framework, con supporto multi-target per .NET Framework e .NET moderno.

## Target Framework

- .NET Framework 4.8
- .NET 8.0
- .NET 8.0-Windows

## Dipendenze Principali

- **Wisej-3** - Framework per applicazioni web progressive
- **Dapper** - Micro ORM per accesso ai dati
- **MiniExcel** - Gestione file Excel
- **Microsoft.VisualBasic** - Supporto runtime

## Installazione

```bash
dotnet add package Passero.Framework.Controls
```

Oppure tramite NuGet Package Manager:

```powershell
Install-Package Passero.Framework.Controls
```
