# Passero.Framework.BusinessSystem

Libreria di supporto ERP base per Passero Framework.

## Descrizione

Passero.Framework.BusinessSystem fornisce le funzionalità di base per sistemi ERP (Enterprise Resource Planning) all'interno del Passero Framework, con supporto multi-target per .NET Framework e .NET moderno.

## Target Framework

- .NET Framework 4.8
- .NET 8.0

## Installazione

```bash
dotnet add package Passero.Framework.BusinessSystem
```

Oppure tramite NuGet Package Manager:

```powershell
Install-Package Passero.Framework.BusinessSystem
```
